GOLF HOLE IMAGES FOLDER
======================

This folder should contain the hole images for your golf course.

Expected file names:
- Hole 1.png
- Hole 2.png  
- Hole 3.png
- Hole 4.png
- Hole 5.png
- Hole 6.png
- Hole 7.png
- Hole 8.png
- Hole 9.png
- Hole 10.png
- Hole 11.png
- Hole 12.png
- Hole 13.png
- Hole 14.png
- Hole 15.png
- Hole 16.png
- Hole 17.png
- Hole 18.png

Instructions:
1. Add your hole images to this folder
2. Make sure the file names exactly match the format above (including the space between "Hole" and the number)
3. Supported formats: PNG, JPG, JPEG
4. The web app will automatically load these images when you select a hole and click the image button

Note: If an image is missing, the app will show a placeholder message instead.
